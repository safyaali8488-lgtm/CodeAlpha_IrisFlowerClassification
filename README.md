# 🌸 Iris Flower Classification

A Machine Learning project developed for the **CodeAlpha Data Science Internship**. This project uses the **Random Forest Classifier** to classify Iris flowers into three different species based on their sepal and petal measurements.

---

## 📌 Project Overview

The Iris dataset is one of the most popular datasets in machine learning. The goal of this project is to build a classification model that accurately predicts the species of an Iris flower using its physical characteristics.

The three flower species are:

- 🌸 Iris-setosa
- 🌼 Iris-versicolor
- 🌺 Iris-virginica

---

## 📂 Dataset

**Dataset:** `Iris.csv`

### Features

- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

### Target Variable

- Species
