# Iris Flower Classification

## Description
This project classifies Iris flowers into three species using a Random Forest Classifier.

## Dataset
- Iris.csv

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Model
Random Forest Classifier

## Accuracy
100%

## Output
The model predicts:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

A confusion matrix is also generated to evaluate the model.
